## Basic details

### Extension name
Hover Image Preview

### Short description
Preview linked images on hover and define selected words with synonyms and antonyms in a clean popup.

### Detailed description
Hover Image Preview helps you browse faster by showing linked image previews on hover and giving instant word definitions from right-click selection.

Features:
- Hover image preview for link targets that are images.
- On-demand word definitions from selected text.
- Synonyms and antonyms when available.
- Source links for additional reference.
- Settings/history stored in Chrome extension storage and can sync across signed-in Chrome profiles.

## Single purpose
This extension provides two productivity features: hover image previews for links and on-demand word lookup popups.

## Permissions justifications

### contextMenus
Adds a right-click context menu item to define selected words.

### storage
Stores extension settings and lookup history and enables sync via Chrome storage when user sync is enabled.

### Host permission: <all_urls>
Required so the extension can detect hovered links and display image previews/definition popup on pages the user visits.
